Project

NEXUS: Streamlining Freelance Payments in Ethiopia

Developed a web platform integrating a secure wallet system for seamless account-to-account transfers and withdrawals to local banks, addressing payment gateway limitations. Features include knowledge sharing and communication among users.
Technologies: HTML, CSS, JavaScript, PHP, MySQL, XAMPP
  • Designed and implemented a secure wallet system (PHP, MySQL).
  • Developed the front-end user interface (HTML, CSS, JavaScript) for all user functionalities.
  • Implemented back-end logic (PHP) for authentication, transactions, and features.
  • Utilized GitHub for version control.
Challenge/Solution: Refactored the chat system's session management to integrate with NEXUS user authentication.
Validation: Reviewed and validated by university faculty.
